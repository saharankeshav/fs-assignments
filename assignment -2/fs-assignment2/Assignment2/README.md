# Assignment News App

Simple React Native (Expo) app that:

- Calls the News API using Axios
- Displays news **title** and **image** in a `FlatList`

## Setup

1. Install dependencies:

```bash
npm install
# or
yarn
```

2. Put your NewsAPI key in `src/api/newsApi.js`:

```js
const API_KEY = 'YOUR_API_KEY_HERE';
```

3. Run the app:

```bash
npm start
# or
yarn start
```

Then open it on Android/iOS/emulator using Expo.
